# Meeting Translator - Dependencies
# 实时会议翻译系统依赖包

# ========================================
# Installation Instructions / 安装说明
# ========================================

# Option 1: Install core dependencies (Aliyun, OpenAI)
# 选项 1：安装核心依赖（阿里云、OpenAI）
#   pip install -r requirements-core.txt

# Option 2: Install core + Doubao dependencies
# 选项 2：安装核心依赖 + 豆包依赖
#   pip install -r requirements-core.txt
#   pip install -r requirements-doubao.txt

# Option 3: Install ALL dependencies (all providers)
# 选项 3：安装完整依赖（所有提供商）
#   pip install -r requirements-all.txt

# ========================================
# Quick Start / 快速开始
# ========================================

# For Aliyun Qwen (default):
# 阿里云 Qwen（默认）:
#   pip install -r requirements-core.txt

# For Doubao (ByteDance):
# 豆包（字节跳动）:
#   pip install -r requirements-core.txt
#   pip install -r requirements-doubao.txt

# For OpenAI Realtime:
# OpenAI Realtime:
#   pip install -r requirements-core.txt

# ========================================
# Requirements Files / 依赖文件说明
# ========================================
# - requirements-core.txt    : Core dependencies (all providers)
#                            核心依赖（所有提供商）
# - requirements-aliyun.txt  : Aliyun-specific (empty, uses core only)
#                            阿里云特定依赖（空，仅使用核心依赖）
# - requirements-doubao.txt  : Doubao-specific (protobuf)
#                            豆包特定依赖（protobuf）
# - requirements-openai.txt  : OpenAI-specific (empty, uses core only)
#                            OpenAI 特定依赖（空，仅使用核心依赖）
# - requirements-all.txt     : All dependencies combined
#                            所有依赖合并
